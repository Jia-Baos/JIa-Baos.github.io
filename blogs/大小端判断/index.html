大小端判断

大端：较高的有效字节存放在较低的存储器地址，较低的有效字节存放在较高的存储期地址；

小端：较高的有效字节存放在较高的存储期地址，较低的有效字节存放在较低的存储期地址。

#include <iostream>

union MyUnion
{
	int a;
	char b;
};

int func()
{
	MyUnion union1;
	union1.a = 1;

	// 因小端存储，所以 union1.a == union1.b
	std::cout << sizeof(union1) << std::endl;
	return union1.a == union1.b;
}

int main(int argc, char* argv[])
{
	if (func())
	{
		puts("小端存储");
	}
	else
	{
		puts("大端存储");
	}

	return 0;
}
