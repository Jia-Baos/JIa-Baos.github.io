常量指针和指针常量

PUBLISHED ON JUL 31, 2023 / 1 MIN READ — ARTICLES

常量指针：指向常量的指针，将指针的内容作为地址寻址，获取该地址处的变量，该变量被const修饰；

指针常量：指针是常量，当前指针被const修饰，将其内容作为地址寻址，获取该地址处的变量，该变量可以变化。

#include <bits//stdc++.h>

int main() {
    int val1{};
    int val2{};

    // 常量指针
    // 可更改指向，但不可更改原指向的内容
    const int *ptr1 = &val1;
    //*ptr1 = 1; // Error
    ptr1 = &val2;

    // 指针常量
    // 可更改指向的内容，但不可更改指向
    int *const ptr2 = &val1;
    *ptr2 = 1;
    //ptr2 = &val2;   //Error
}
